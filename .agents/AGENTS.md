# Project Terminology Standards

When editing code, writing comments, creating APIs, or adding new features in this repository, always align terminology with standard Australian maritime and port operations conventions.

## Terminology Dictionary

| Legacy Term | Standardised Term | Definition / Scope |
| :--- | :--- | :--- |
| **Shift** / **Transit** | **Shifting** | Vessel movements between berths within Newcastle Harbour. |
| **Voyage** | **Port Call** | The overall visit of a vessel to the port (Arrival $\rightarrow$ Departure). |
| **Stay** / **Visit** | **Berth Occupancy** / **Time on Berth** | The specific period a vessel spends moored at a particular berth. |
| **Stay Efficiency** | **Time on Berth Efficiency** | Analysis comparing actual time on berth against planned time on berth. |
| **Voyage counts (KPIs)** | **Movements** | Count of specific arrival/departure actions (e.g. "Incoming Movements" / "Outgoing Movements"). |
